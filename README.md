# node_ed6 boilerplate

### features
 - ES6 support
 - ESlint google rule
 - Dockerlize
 - jest unit test

### start project

install dependencies by `yarn` or `npm install`

```
npm install
```

To run this project in development mode

```
npm run dev
```

build project and start it on production mode

```
npm run build && npm start
```
